## 使用说明

### claude-udp-reject
用于拒绝 Claude.ai 和 Anthropic.com 的 UDP 连接的 Clash 分流规则。

**GitHub Raw 链接**:
```
https://raw.githubusercontent.com/joeyliu6/MyConfig/master/claude-udp-reject.yaml
```

**jsDelivr CDN 链接（国内推荐）**:
```
https://cdn.jsdelivr.net/gh/joeyliu6/MyConfig@master/claude-udp-reject.yaml
```

**使用方法**:
1. 复制上述链接
2. 在 Clash 客户端中添加远程规则
3. 将匹配动作设置为 REJECT
